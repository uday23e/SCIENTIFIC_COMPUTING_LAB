# Generated Plot Folder

Run

```bash
python optimization_methods.py
```

to generate the following SVG files in this directory:

- `steepest_descent_contour.svg`
- `modified_newton_contour.svg`
- `iterate_paths_comparison.svg`
- `convergence_history.svg`

These plots visualize the iterate paths and contour-based convergence behavior for Steepest Descent and the Modified Newton method.
